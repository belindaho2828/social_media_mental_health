# social_media_mental_health
An analysis of the impact of social media usage on young people in 2022 (Bangladesh).
Original dataset downloaded from https://www.kaggle.com/datasets/souvikahmed071/social-media-and-mental-health.
Analysis was performed by Meredith Jones, Celina Kamler, and Belinda Ho.
This assignment was completed as a requirement for the UC Berkeley Data Analytics & Data Visualization Boot Camp.
Extra care was taken to ensure we conducted original analysis. Our team focused on exploring the following research question: Does social media usage impact mental health differently for women?
